# Module: Database Design and Implementation  
## Assignment Title: Healthcare Appointment and Patient Records System

---

## Overview
This project implements a relational healthcare database system using MySQL 8+.  
It models patients, doctors, facilities, treatments, and appointments while enforcing referential integrity and normalization principles.

---

## Project Structure

- `healthcare_system.sql`  
  Full MySQL script including:
  - Database creation
  - Table definitions
  - Constraints
  - Sample data
  - Business queries
  - CASE + GROUP BY analysis

- `report/assignment_writeup.md`  
  Written assignment (conceptual explanations, ER modelling, normalization discussion).

- `screenshots/`  
  Proof of execution in MySQL Workbench.

---

## How to Run

1. Open MySQL Workbench.
2. Execute `healthcare_system.sql`.
3. Run the queries.
4. Verify results using screenshots.
